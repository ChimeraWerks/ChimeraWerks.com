01

One console, whole fleet

×Every agent you add is another terminal to babysit, another transcript scrolling where you are not looking.

→Relay puts the fleet in shared rooms — personas, live transcripts, spawn controls. However many agents run, you watch one console.

02

Orchestration you can diff

×Judge panels, fusion, debates — in most shops that logic lives in glue scripts only the author can rerun.

→Relay speaks Chimera DNA, a declarative run language. The whole orchestration becomes a document: readable, diffable, rerunnable on any harness.

03

Failures keep their trail

×When an agent dies three hops down, most consoles hand you a normalized reason code and eat the stderr.

→Relay keeps the raw trail: child stderr, RPC hops, exit codes, secrets redacted. The real error is one layer down, never gone.
